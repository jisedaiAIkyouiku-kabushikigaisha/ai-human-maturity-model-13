# AI人材成熟モデル13段階・現在地確認フレーム

このリポジトリは、AI活用の状態を13段階で整理した「AI人材成熟モデル13段階・現在地確認フレーム」の原典管理用リポジトリです。

本モデルは、AI人材を一方的に格付けするためのものではありません。

「AI人材」「AIネイティブ」「プロンプト人材」「AIを使いこなす人」といった言葉が広がる一方で、現場では、個人利用・業務化・組織実装・社会実装の違いが曖昧になりやすくなっています。

そのため、本モデルではAI活用の現在地を段階的に整理し、受講者・企業・講師・支援者が、次に必要な学習内容や支援方針を確認しやすくすることを目的としています。

## 目的

このモデルの目的は、AI活用を「できる・できない」で単純に分けることではありません。

現在どの範囲まで扱えているのか、次にどの力を伸ばすべきなのか、個人利用で止まっているのか、業務化できているのか、組織に実装できているのかを整理するための参照フレームです。

日本社会では、段階や目安がないと成長の方向性をイメージしにくい場面があります。偏差値や等級のような比較文化が強い一方で、それを単なる格付けとして扱うと、成長支援ではなく評価圧力になってしまいます。

本モデルでは、段階を「上下を決めるための序列」ではなく、「現在地と次の成長方向を確認するための地図」として扱います。

## 13段階の全体像

| 段階 | 呼び方                       | できること                      |
| -: | ------------------------- | -------------------------- |
|  1 | AI User                   | AIを日常的に使える                 |
|  2 | Prompt Operator           | 目的に合わせて指示を組める              |
|  3 | AI Power User             | 複数用途でAIを使い倒せる              |
|  4 | Workflow Builder          | AIを業務手順に組み込める              |
|  5 | AI Orchestrator           | AI・人・ツール・データを束ねて成果物を出せる    |
|  6 | AI Architect              | AI前提で業務・組織・データを設計できる       |
|  7 | AI Native                 | 思考・判断・実行がAI前提になっている        |
|  8 | AI Model Architect        | AIそのものを設計できる               |
|  9 | AI Paradigm Builder       | AI時代の新しい前提・産業・文化を作れる       |
| 10 | AI Ecosystem Designer     | AIが自然に回る経済圏・教育圏・組織圏を設計できる  |
| 11 | AI Civilization Architect | AIを前提に社会制度・働き方・学び方まで再構築できる |
| 12 | AI Meta Designer          | AI・人間・社会の設計思想そのものを設計できる    |
| 13 | AI Origin                 | 次のAI Userを生み出す起点になる        |

## 輪廻構造

このモデルでは、13段階目を単なる終点とは考えていません。

AI Originは、自分が作ったAI、教材、研修、仕組み、文化、プロダクト、組織によって、新しいAI Userを生み出す起点です。

つまり、AI Userが成長してAI Originになり、AI Originが次のAI Userを生み出すという循環構造を持ちます。

## 位置づけ

本モデルは、固定された絶対基準ではありません。

AI技術、社会制度、教育現場、企業導入の変化に合わせて、今後も見直し・改善される可能性があります。

そのため、本リポジトリではバージョン管理を行い、変更履歴を残します。
